Правила користування АСВП-ботом

Можливі формати для обробки (EXCEL):

Пошук по АСВП:

  • Необхідні колонки:
    • 'Номер ВП'

Пошук по ПІБ:

  • Необхідні колонки:
    • 'INN' - ІПН,
    • 'FIO' - ПІБ,
    • 'BirthdayDate' - дата народження

  • Формат 'BirthdayDate': 'Текст' в форматі (%dd-%mm-%YYYY) (наприклад: '17-08-2020') або 'Краткий формат даты'
  • BirthdayDate можна вирахувати з ІПН за допомогою формули '=ЛЕВСИМВ(#INN;5)+1'

Пошук по ІПН (Єдиний реєстр боржників + АСВП):

  • Необхідні колонки:
    • 'INN' - ІПН

Пошук по ЄДРПОУ:

  • Необхідні колонки (має бути щонайменше одна з них):
    • 'ЄДРПОУ' - ЄДРПОУ,
    • 'Назва' - Назва компанії,

Пошук в "Доступ сторін до виконавчого провадження":

  • Необхідні колонки:
    • 'Номер ВП'
    • 'Ідентифікатор доступу'

Завантаження файлів з АСВП

  • Необхідні колонки:
    • 'Номер ВП'
    • 'Ідентифікатор доступу'
    • 'Назва постанови' (для завантаження файлів за назвою постанови)
    • '№ з/п' або 'ID' (для завантаження файлів за ID з АСВП)
  • Додаткові колонки:
    • 'Шлях' (шлях для папки, в яку необхідно зберегти файли; якщо шлях однаковий для всіх файлів - можно вказати через інтерфейс при запуску)
    • 'Назва файлу' (якщо не вказано - назва файлу відповідатиме назві постанови)
