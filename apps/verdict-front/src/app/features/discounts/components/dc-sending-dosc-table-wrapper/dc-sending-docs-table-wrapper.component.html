Завантаження...
Етап 1: Завантаження Документів

Виберіть спосіб відправки документів із запропонованих опцій. Виділіть конкретні заяви для завантаження або оберіть усі відразу. Натисніть кнопку "Завантажити документи", щоб отримати zip-архів із необхідними документами.

Етап 2: Підтвердження Відправки

Використовуйте фільтр для вибору методу відправки. Якщо вибрано спосіб відправки "Електронна пошта", достатньо лише вказати дату відправки. Для інших способів відправки у розділі "Дата відправки" прикріпіть файл підтвердження відправки, зазначивши тип документу і введіть дату відправки.

Етап 3: Перегляд Заяв

Перегляньте всі заяви, по яких були завантажені та підтверджені документи. Використовуйте цей дашборд для ефективного управління процесом відправки та контролю за заявами.
